# eigen

A modern, pixel-perfect PDF editor/viewer built with Electron, Vite, Vanilla JS, and PDF.js. Edge-like UI/UX. Fast, professional, and minimal.

- Multi-tab, browser-like PDF experience
- High-quality rendering, dark mode, annotation tools
- Optimized for large PDFs

## Usage

1. `npm install`
2. `npm run dev` (for browser)
3. `npm run electron` (for Electron app)
